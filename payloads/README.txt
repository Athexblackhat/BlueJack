REM ============================================
REM BlueJack Payload Examples
REM ============================================
REM 
REM Place any .txt file containing DuckyScript in this folder
REM The tool will automatically list and allow selection
REM 
REM Supported Commands:
REM 
REM DELAY <ms>           - Wait for specified milliseconds
REM STRING <text>        - Type the specified text
REM ENTER                - Press Enter key
REM TAB                  - Press Tab key
REM GUI r / CTRL c       - Modifier key combinations
REM REM <comment>        - Comment line (ignored)
REM PRIVATE_BROWSER      - Opens private/incognito window (Ctrl+Shift+N)
REM VOLUME_UP            - Increases volume (requires media keys)
REM 
REM Supported Modifiers:
REM - GUI (Windows/Command key)
REM - CTRL
REM - ALT
REM - SHIFT
REM 
REM Example combinations:
REM GUI r                 - Open Run dialog (Windows)
REM CTRL c                - Copy
REM CTRL ALT DELETE       - Task Manager (may not work via HID)
REM ALT TAB               - Switch windows
REM 
REM Note: Some combinations may behave differently across operating systems
REM ============================================
